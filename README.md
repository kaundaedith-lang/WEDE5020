# Childrens Hospital Foundation Website

This is a multi-page website for a nonprofit organization, inspired by real nonprofit websites such as [Children’s Hospital Trust](https://www.childrenshospitaltrust.org.za/).


##  Project Structure
nonprofit-website/
│── index.html       # Home page
│── about.html       # About Us page
│── programs.html    # Programs / What We Do page/Donnation
│── contact.html     # Contact page
│── styling.css     # stylesheet

│
├── images/      # Placeholder for images (logo, banners, etc.)
└── js/          # Placeholder for JavaScript files


## Features
- Clean and simple *navigation bar* across all pages  
- *Home page* with a hero section and quick links  
- *About page* explaining the organization’s mission, vision, and values  
- *Programs page* with different ways to contribute  
- *Contact page* with an inquiry form and contact details  
- Consistent header layout with logo and title  
- Organized project structure for future expansion (JS)


## Styling Updates
- Integrated a custom **color palette** featuring warm tones: yellow, orange, beige, and earthy brown  
- Added a **recolored logo** to match the palette and placed it next to the site title in the header  
- Styled buttons with orange and yellow for clear call-to-action emphasis  
- Applied consistent typography and spacing for improved readability  
- Used Flexbox for responsive header layout and logo positioning  
- Enhanced link styles with hover effects for better user interaction

## How to Use
1. Download or clone the project  
2. Open `index.html` in any modern web browser  
3. Navigate between pages using the menu links  
4. View styling by inspecting the layout and color scheme across pages

## Notes
- JavaScript functionality is not yet implemented  
- The *donation button* is a placeholder and does not process real payments
